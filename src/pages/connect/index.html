Connect your starter kit in three easy steps

Your computer will relay the sensor data to the Pareto cloud platform.

Not your starter kit? Click here.

1. Install Node.js

Install Node.js on the computer to which your starter kit will be connected. We recommend you install the Long Term Support version.

Let's do it!

2. Install barnowl-relay

Once the Node.js installation completes, open a terminal or command prompt and enter:

npm install -g barnowl-relay

3. Connect and run

With the starter kit plugged in to the USB port of your computer, enter via the terminal:

barnowl-relay -sv

Your computer is now relaying the sensor data to the Pareto cloud platform. Log in and begin your trial!
