    For Developers

    Our Value Proposition

    To developers of software and systems for physical world applications

    It is increasingly common for initiatives to involve some combination of IoT, real-time location, RFID and/or wireless sensing. Pareto Anywhere provides you a versatile open source framework so you can invest your time into developing the application itself without having to divest your time into developing the underlying layers.*

    *Of course, if you prefer developing those underlying layers, we'd welcome your contributions!

    By developers...

    ...for developers

    Pareto Anywhere is the third iteration of our open source code base and, as developers ourselves, we've (selfishly) taken initiatives to make this the most developer-friendly yet. We're proud that Pareto Anywhere is:

      Open source & MIT-licensed
    Find on GitHub the code you need and do with it what you need!
      Cost-free
    Use the software at no cost and upgrade to an annual subscription only once the added benefits make sense.
      Documented (mostly!)
    We strive to set the bar high for our tutorials on reelyactive.github.io as we depend on both them and our project README files ourselves!
      Vanilla JavaScript
    We've done away with frameworks to keep it light and simple enough for even a novice to comfortably dive in.
